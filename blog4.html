Cara merubah warna dan lebar scrollbar pada website menggunakan CSS

Scrollbar adalah penggeser yang digunakan untuk melihat bagian bawah suatu blog atau website.

Setiap browser memiliki default scrollbarnya sendiri. Biasanya, berwarna abu - abu tua dan berada pada sisi sebelah kanan sebuah browser.

Tapi, pernahkah kamu melihat suatu website yang memiliki scrollbar berwarna biru atau merah ? Atau pernahkah kamu melihat scrollbar disuatu website yang mempunyai lebar tidak seperti biasanya ?

Walaupun setiap browser sudah menerapkan bentuk scrollbarnya sendiri, kita masih bisa merubah warna dan bentuk scrollbar tersebut.

Ada beberapa cara merubah style pada scrollbar. Tetapi kali ini, cara yang saya bagikan adalah dengan menggunakan attribute pada CSS.

"Hidup itu sederhana, kita yang membuatnya sulit."

- Confucius

Pertama, buka code editor kamu, lalu buat file index.html baru dan style.css baru

Buka index.html dan tulis sintaks default html.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="style.css">
    <title>Mengubah Scrollbar</title>
</head>
<body>
    
</body>
</html>

Dan jangan lupa untuk menghubungkan file index.html dengan style.css

Setelah itu, buka style.css dan terapkan style berikut pada selector "body".

body { 
    height : 200vh;
}

Style tersebut untuk merubah tinggi dari body, yang dimana bertujuan agar kita dapat melihat scrollbar default browser.

1. Jika kamu menggunakan google chrome sebagai browser, maka tambahkan style berikut pada css

Pertama kita ubah dahulu lebar dari scrollbar

body::-webkit-scrollbar{
    width : 7px;
}

Setelah itu, kita dapat mengubah warna track yang ada pada bagian scroll

body::-webkit-scrollbar-track{
    background-color : #1f1f1f;
}

Dan yang terakhir, kita ubah warna thumb yang ada pada scrollbar

body::-webkit-scrollbar-thumb{
    background-color : lightblue;
}

Hasil

2. Jika kamu menggunakan firefox sebagai default browser, maka gunakan cara ini 

:root{
    scrollbar-color: #2aadeb #1f1f1f;
    scrollbar-width: thin !important;
}

Kedua cara diatas memiliki fungsi yang sama, hanya saja cara pertama hanya bisa bekerja di browser google chrome dan cara kedua hanya bekerja pada browser firefox.

Tentu, kamu dapat bermain dengan property - property diatas. Ubah sesuai selera atau tema website yang kamu miliki.
